# ImoTech

Aplicação web para uma imobiliária 

O ituito era mostrar como funciona o React js

## Tecnologias
 - React js
 - Vite js
 - Tailwindcss
 - Json server
 - Lucide icons
 - Axios
 - React Router Dom

 ## Para rodar a aplicação

 ```bash
  // frontend
  npm run dev

  // rodar json server
  npm run dev:server
 ```
